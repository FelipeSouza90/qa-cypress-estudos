# Projeto 02 - Testes básicos de formulário

Projeto de estudos com Cypress para praticar testes automatizados em elementos comuns de formulário e interface.

## Objetivo

Aprender a testar campos de entrada, visibilidade de elementos e cliques em botões.

## Funcionalidades testadas

- digitação em campo de e-mail
- digitação em outro campo de formulário
- validação de visibilidade de campo
- validação de visibilidade de botão
- clique em botão

## Comandos praticados

- `cy.visit()`
- `cy.get()`
- `cy.type()`
- `cy.click()`
- `cy.should()`

## Arquivo principal

- `formulario-basico.cy.js`

## Aprendizados

Neste projeto foram praticados:
- preenchimento de formulário
- validação de valor com `have.value`
- verificação de visibilidade com `be.visible`
- interação com botão via `click()`
