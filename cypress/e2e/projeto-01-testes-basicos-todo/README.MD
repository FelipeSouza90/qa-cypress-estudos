# Projeto 01 - Testes básicos da aplicação Todo

Projeto de estudos com Cypress para praticar testes automatizados básicos em uma aplicação de lista de tarefas.

## Objetivo

Praticar os comandos iniciais do Cypress e validar comportamentos simples de interface.

## Funcionalidades testadas

- adicionar uma nova tarefa
- filtrar tarefas ativas
- filtrar tarefas concluídas

## Comandos praticados

- `cy.visit()`
- `cy.get()`
- `cy.contains()`
- `cy.type()`
- `cy.click()`
- `cy.check()`
- `cy.should()`

## Arquivo principal

- `todo-basico.cy.js`

## Aprendizados

Neste projeto foram praticados:
- abertura de página
- busca de elementos por seletor
- busca de elementos por texto
- interação com campos e filtros
- validação de resultados esperados
