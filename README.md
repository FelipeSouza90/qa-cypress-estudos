# QA Cypress Estudos

Repositório criado para estudar testes automatizados com Cypress, com foco em aprendizado prático e construção de portfólio para a área de tecnologia.

## Objetivo

Aprender a criar testes automatizados para aplicações web, simulando o comportamento de um usuário no navegador e validando fluxos importantes da interface.

## Conteúdo estudado até agora

- Instalação e configuração do Cypress
- Estrutura inicial do projeto
- Execução de testes na interface visual do Cypress
- Comandos básicos:
  - `cy.visit()`
  - `cy.get()`
  - `cy.contains()`
  - `cy.type()`
  - `cy.should()`

## Primeiro teste criado

Teste inicial criado para praticar:
- abertura de página
- interação com campo de texto
- adição de nova tarefa
- validação de quantidade de itens

## Tecnologias utilizadas

- Node.js
- npm
- Cypress
- JavaScript
